#### Add a WAF to the Ingress

Create a WAF ACL
```Bash
WAF_WACL_ARN=$(aws wafv2 create-web-acl --name <acl-name> --region <region-id> --default-action Allow={} --scope REGIONAL --visibility-config SampledRequestsEnabled=true,CloudWatchMetricsEnabled=true,MetricName=YelbWAFAclMetrics --description "<waf-description>" --query 'Summary.ARN' --output text )
```

Get the ID of the ACL
```Bash
WAF_WAF_ID=$(aws wafv2 list-web-acls --region <region-id> --scope REGIONAL --query "WebACLs[?Name=='<acl-name>'].Id" --output text)
```

#### Update the Ingress to use the WAF

Add an annotation to the ingress manifest that points to the ACL ARM alb.ingress.kubernetes.io/wafv2-acl-arn
E.g.
```yaml
apiVersion: networking.k8s.io/v1
kind: Ingress
metadata:
  name: nginx-ingress
  annotations:
    alb.ingress.kubernetes.io/scheme: internet-facing
    alb.ingress.kubernetes.io/target-type: ip
    alb.ingress.kubernetes.io/wafv2-acl-arn: arn:aws:wafv2:eu-central-1:068691969032:regional/webacl/WAF-FOR-NGINX/547d15e7-009e-4610-84a6-6f317e16e2ab
spec:
  ingressClassName: alb
  rules:
  - http:
      paths:
      - path: /
        pathType: Prefix
        backend:
          service:
            name: nginx-service
            port:
              number: 80
```

#### Enable Traffic Filtering

Create a rules file in json format.
This rules file utilizes the AWS Bot Control to block bots
```json
[
    {
      "Name": "AWS-AWSManagedRulesBotControlRuleSet",
      "Priority": 0,
      "Statement": {
        "ManagedRuleGroupStatement": {
          "VendorName": "AWS",
          "Name": "AWSManagedRulesBotControlRuleSet"
        }
      },
      "OverrideAction": {
        "None": {}
      },
      "VisibilityConfig": {
        "SampledRequestsEnabled": true,
        "CloudWatchMetricsEnabled": true,
        "MetricName": "AWS-AWSManagedRulesBotControlRuleSet"
      }
    }
]
```

Update the ACL with the new rules file
```Bash
aws wafv2 update-web-acl --name <acl-name> --scope REGIONAL --id $WAF_WAF_ID --default-action Allow={} --lock-token $(aws wafv2 list-web-acls --region <region-id> --scope REGIONAL --query "WebACLs[?Name=='<acl-name>'].LockToken" --output text) --visibility-config SampledRequestsEnabled=true,CloudWatchMetricsEnabled=true,MetricName=YelbWAFAclMetrics --region <region-id> --rules file://<rules-file>
```
