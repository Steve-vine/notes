# Ubuntu Common Bash Commands

---
Show IP addresses
```Bash
ip addr show
```

ls Switches
| -G | Show Colours |
| -F | Add slashes to folders |
| -l | Long listing (size, owner dates etc.) |
| -a | All files including hidden files |
| -s | Show size in blocks not bytes |
| -h | Show size in human readable format |
| -R | Recursively list subdirectories |
