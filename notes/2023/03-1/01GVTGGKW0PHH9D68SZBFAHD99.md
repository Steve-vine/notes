# Basic git Commands

---
https://www.freecodecamp.org/news/10-important-git-commands-that-every-developer-should-know/

### Push changes to github

Get status
```Bash
git status
```

Add file(s)
```Bash
get add -A
```

Commit them
```Bash
git commit -m 'comments'
```

Push them up
```Bash
git push
```

### Create a new git repo

Convert a folder into a repo
```Bash
git init
```

Add Files
```Bash
git add -A
```

Commit them
```Bash
git commit -m 'comments'
```

Configure remote repo
```Bash
git remote add origin <url>
```

Push the files and specify the branch
```Bash
git push -u origin main
```
