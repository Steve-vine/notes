# Basic Commands

Start Minikube
```Bash
minikube start
```

Start Minikube with resource parameters
```
minikube start --cpus 8 --memory 8g
```

Set resource usage, prior to starting
```Bash
minikube config set cpus 4
minikube config set memory 6114
```

Mount a local folder into Kubernetes
```Bash
minikube mount \Users\Steve\Code\ismsct-app\www:/localcode
```

# Port Forwarding and Tunnelling

Create a tunnel to a service and open it in the browser
```Bash
minikube service <servicename>
``` 

Forward a port 
```Bash
kubectl port-forward service/ismsct-nginx 8080:80
```

# Profiles

Start Minikube with a specific profile
```
minikube start --profile=mycluster
```

Stop a specific profile
```
minikube stop --profile=mycluster
```

Delete a profile
```
minikube delete --profile=mycluster
```

List all profiles
```
minikube profile list
```

Switch profiles
```
minikube profile mycluster
```
