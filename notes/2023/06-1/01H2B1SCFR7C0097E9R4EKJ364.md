#### Provider
Add-in that provides a set of CRD's used to manage infrastructure e.g. 'provider-aws'

#### Custom Resource Definition (CRD)
A single resource definition that manages a component e.g. 'S3 Bucket'

#### Managed Resources (MR)
Resources managed by Crossplane as CRD's supplied through a provider

#### Composite Resource (XR)
One or more Managed Resources

#### Composite Resource Definition (XRD)
Specifies the schema for a Composite Resource, cluster scoped

#### Composite Resource Claim (XRC)
Facilitates the creation of a namespace
