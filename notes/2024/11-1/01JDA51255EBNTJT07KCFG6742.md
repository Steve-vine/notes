### Overview
There are two AWS account:
ECR Account - This contains the ECR Image to be pulled
Cluster Account - This account contains an EKS Management Cluster that needs to authenticate with ECR in the first account to deploy the image

### Cluster Account
#### 1. Create IAM Role
Create an IAM role to be used by pods in the Cluster
Name: **ecr-image-pull-role**

#### Create Trust Policy
Create a trust policy as below
Name: **ecr-image-pull-policy**
```
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Principal": {
        "Federated": "arn:aws:iam::<cluster-account-id>:oidc-provider/<oidc-provider-url>"
      },
      "Action": "sts:AssumeRoleWithWebIdentity",
      "Condition": {
        "StringEquals": {
          "<oidc-provider-url>:sub": "system:serviceaccount:<namespace>:<service-account-name>"
        }
      }
    }
  ]
}
```

#### 2. Create Permissions Policy
Attach ECR permissions for cross-account access
```
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "ecr:GetAuthorizationToken",
        "ecr:BatchCheckLayerAvailability",
        "ecr:GetDownloadUrlForLayer",
        "ecr:BatchGetImage"
      ],
      "Resource": "*"
    }
  ]
}
```
#### 3. Create a Kubernetes Service Account
Create a service account in the EKS cluster and annotate it with the ARN of the IAM role from Step 1

Trust Policy:
```
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Principal": {
        "AWS": "arn:aws:iam::<cluster-account-id>:role/<eks-role-name>"
      },
      "Action": "sts:AssumeRole"
    }
  ]
}
```

### ECR Account
#### Grant Assume Role Permissions
Create an IAM role in the **ECR Account** that allows the **Cluster Account’s IAM role** to assume it
Name: ecr-image-pull-role

### Create Trust Policy
Create a trust policy as below
Name: **ecr-image-pull-policy**
```
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "ecr:GetAuthorizationToken",
        "ecr:BatchCheckLayerAvailability",
        "ecr:GetDownloadUrlForLayer",
        "ecr:BatchGetImage"
      ],
      "Resource": "*"
    }
  ]
}
```
