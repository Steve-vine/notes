I would like to redesign the UI - Below are some mock-ups to give you an idea of what I want.

Key points:

* Cleaner, simpler design, simple icons
* Left and right panels, both collapsible
* Within each panel the data sections are collapsible (represented by the lighter header)
* New main panel tabs at top of screen.  User can add different tabs to have different views open and switch between them
* "Type of view" tab bar moved to the main panel, 'search' and 'browse' now single tab, collapsible sections in left bar move between the browse/search.  New Dashboard tab added, will expand on this later.

Browse Mode

* While in Browse mode, can now add multiple taxonomy types to the left pane, which show nested in the order selected.
* Note taxonomies now moved from the note header to the right hand panel
* When splitting the note panels, it should just show as multiples of the grey panel, no borders, the gap between the panels is the drag/resize zone.  Text on non-active panes should be slightly darker (lighter in light mode) to show as inactive.

Kanban Mode

* Kanban mode, resize columns by clicking and dragging in the gap between them, no highlight appears
* Revamped kanban card, cleaner style, added due date.  Properties icons -
  * Priority (L, M, H, C)
  * Blocked (Dependant on another task being completed)- Will add functionality for this later
  * Blocking (Preventing another task from being created) - Will add functionality for this later
  * More icons to follow later
* Single click on a card to select it and show properties in right hand panel, double click to open card
* When opening the card, stay in Kanban mode but open the card over the top of the Kanban, and add an 'X' in the top left to close the note and return to the Kanban.
