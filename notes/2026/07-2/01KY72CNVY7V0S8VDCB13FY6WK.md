Standardise key presses on the new note window. 

Escape key cancels the window and closes it, unless the main content has been edited, in which case display a modal with a confirmation allowing closure or cancel and go back to edit.

Tab key (or shift-tab to go backwards) moves through the objects - Title, content, Type etc.

Hit enter on an object like a dropdown list displays the content, arrow keys move up and down and enter selects the item and collapses the dropdown.

Alt  + enter saves the note and closes the window

ctrl + alt + enter opens the note in the main screen
