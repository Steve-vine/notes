This is an investigation piece.  I'd like to improve the content management part of the app.  Specifically I want to make it easier, more WYSIWYG, and more flexible.

Challenges

* Most people aren't familiar with markdown
* The markdown doesn't exactly match the generated PDF output
* TOC isn't easy to achieve
* Other formatting things like tables, images etc aren't easy to achieve
* Sometimes different document types might be needed (E.g. spreadsheets)

Ideal scenario

* Upload any download any type of document into the app
* Create shortcuts to external documents in the app
* Select a domain and see a list of all the uploaded documents associated with it
* Click to view documents in the web app (or follow link to external documents).
* Edit documents in the app rather than download, change, upload.
* Track document releases in the app
* Inject additional data into a PDF before downloading (Review history)

Some of these things are more complex than other and any solution will include compromises.  I'd like to understand what options are available and talk them through before deciding on a solution.
